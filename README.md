# BrightBox

BrightBox is a collection of utilities you can use on a 32x32 LED matrix!

## Photos

Photos can be found here: https://justinkrumlauf.me/gallery/brightbox_photos

## Configurations
BriteBox can be configured to display:
- Current time
- Current date
